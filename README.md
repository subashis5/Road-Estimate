# 🛣️ Smart Road Intervention Estimator

A modern, eco-friendly web application for estimating costs and materials required for road safety interventions.

## 📋 Project Overview

**Smart Road Intervention Estimator** is a professional web tool designed for engineers, planners, and government agencies to quickly calculate the costs of various road safety interventions. Built for the **National Road Safety Hackathon 2025**, this application combines sustainability principles with practical functionality.

## ✨ Features

### 🎨 Modern Design
- Clean, minimal interface with white and green eco-friendly theme
- Glassmorphism effects and smooth animations
- Fully responsive design (works on desktop, tablet, and mobile)
- Dark/Light mode toggle with localStorage persistence

### 📊 Cost Estimation
- Support for 6 types of interventions:
  - Speed Breaker
  - Guard Rail
  - Road Signage
  - Street Light
  - Zebra Crossing
  - Traffic Signal
- Multiple material options (Concrete, Asphalt, Metal, Plastic, Composite)
- Location-based calculations (Urban, Rural, Highway, Residential)
- Real-time cost breakdown with visual charts

### 🤖 AI Recommendations
- 24 unique contextual recommendations
- Based on intervention type and location
- Professional safety and sustainability advice

### 📥 PDF Report Generation
- Professional government-style reports
- Complete with input parameters and cost breakdown
- AI recommendations included
- "Verified by Smart Road Estimator" footer
- Instant download to device

### 📈 Data Visualization
- Interactive pie chart showing Material vs Labor cost breakdown
- Built with Chart.js library

## 🚀 Getting Started

### Installation

1. **Download the Project**
   - Download and extract the `smart-road-estimator.zip` file
   - Or clone the repository

2. **File Structure**
   ```
   smart-road-estimator/
   ├── index.html      (Main HTML structure)
   ├── style.css       (Complete styling)
   ├── script.js       (JavaScript functionality)
   └── README.md       (This file)
   ```

3. **Open the Application**
   - Simply open `index.html` in any modern web browser
   - No server or installation required!

### Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for loading external libraries):
  - Google Fonts (Poppins)
  - Chart.js
  - jsPDF

## 💻 Usage

1. **Navigate to Estimator**
   - Click "Start Estimating" from the home page
   - Or use the navigation menu

2. **Fill in the Form**
   - Select Intervention Type
   - Choose Material Type
   - Enter Quantity
   - Select Location Type
   - Input Labor Cost per Unit (₹)
   - Input Material Cost per Unit (₹)

3. **Calculate**
   - Click "Estimate Now" button
   - View instant results with cost breakdown
   - Read AI-powered recommendations
   - See visual chart representation

4. **Download Report**
   - Click "📥 Download Report as PDF"
   - PDF will be saved to your device
   - Share with stakeholders or attach to proposals

5. **Reset & Repeat**
   - Use "Reset" button to clear form
   - Perform multiple calculations as needed
